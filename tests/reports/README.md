## Report Generation

### Test Reports
Generated by running the unit test suite:
```bash
cd build
./tests/chirp_test
./tests/chirp_factory_test
./tests/chirp_logger_test
```

### Coverage Reports
Generated with coverage instrumentation:
```bash
cd build
cmake .. -DBUILD_TESTS=ON -DENABLE_COVERAGE=ON
make -j$(nproc)
./tests/chirp_test > /dev/null 2>&1
./tests/chirp_factory_test > /dev/null 2>&1
./tests/chirp_logger_test > /dev/null 2>&1
gcov *.gcda
```

### Performance Reports
Generated by running benchmark executables:
```bash
cd build
./tests/chirp_benchmark
./tests/message_throughput_benchmark
```

## Report Maintenance

Reports are automatically generated during the testing and benchmarking process. To regenerate all reports:

1. **Run Tests:**
   ```bash
   cd build
   make clean
   cmake .. -DBUILD_TESTS=ON -DENABLE_COVERAGE=ON
   make -j$(nproc)
   ```

2. **Execute Test Suite:**
   ```bash
   ./tests/chirp_test
   ./tests/chirp_factory_test
   ./tests/chirp_logger_test
   ```

3. **Run Benchmarks:**
   ```bash
   ./tests/chirp_benchmark
   ./tests/message_throughput_benchmark
   ```

4. **Generate Coverage:**
   ```bash
   gcov *.gcda
   ```
